# 🚀 HackXtreme'25

An interactive and informative hackathon website built entirely using Next.js, created to support and engage participants of the HackXtreme'25 event.

## 🌟 Overview

HackXtreme'25 is an informative and interactive website designed to provide participants with all necessary details about the upcoming hackathon event. Participants can view event highlights, rules, timeline, and register for the hackathon through the platform.

## 🔧 Features

* 📅 Event overview and timeline
* 📄 Rulebook and brochure download options
* 📝 Online registration form
* 🎨 Responsive and visually engaging UI
* 🌐 Built entirely using Next.js

## 🛠️ Tech Stack

**Frontend:**

* Next.js
* Tailwind CSS
* CSS Modules

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js (v18+)
* npm

### Local Development

Follow these steps to run the project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/sriram2915/HackXtreme.git
   ```

2. **Install the dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser** and visit `http://localhost:3000` to see the website running locally.
